# 用药助手 (Medication Assistant)

一款采用 Material Design 风格的极简 iOS 用药提醒 APP, 帮助你按时服药, 养成健康习惯。

## ✨ 主要功能

- 📅 **每日用药计划**: 自定义药品名称、剂量、服药时间和有效期
- ✅ **打卡签到**: 一键完成每日服药打卡, 简洁高效
- 🔔 **智能提醒**: 每日定时推送用药通知, 错过打卡30分钟后再次提醒
- 📊 **历史记录**: 日历视图和统计图表, 清晰查看服药历史
- 🔥 **连续打卡**: 记录连续打卡天数, 激励养成好习惯
- 📈 **数据分析**: 总完成率、各药品统计, 直观了解用药习惯

## 🎨 设计风格

- 极简 Material Design 风格
- 卡片化布局, 圆角设计
- 8种 Material 配色可选
- 阴影层级清晰, 视觉舒适
- 状态徽章: 待服用 / 已服用 / 已错过 / 已跳过

## 💡 使用说明

1. **添加药品**: 点击主页或药品页右下角的 + 按钮, 填写药品信息
2. **打卡**: 主页显示今日所有待服用药品, 点击右侧对勾即可打卡
3. **查看历史**: 切换到"记录"标签, 可按日历查看或查看统计
4. **编辑药品**: 在药品列表点击对应药品, 进入详情页编辑
5. **停止/启动**: 编辑药品时可切换"启用"状态

## 📝 数据存储

所有数据保存在 iOS 沙盒的 UserDefaults 中, 通过 JSON 编码持久化, 卸载 APP 时数据会清除。

## 🔒 隐私

- APP 不联网, 不上传任何数据
- 所有数据均保存在设备本地
- 通知仅用于用药提醒, 不会推送广告
